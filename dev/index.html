SeaweedFSClient.save — Function
save(f::NamedTuple, file_data)
save(h::Host, url, fid, file_data)

Arguments

f is the response returned from create (or any tuple with (host=(hostname=String, port=Int), fid="V,FID", url="http...")) Send the file_data to the fid. file_data is created using filedata()

Example

# using a created f
julia> save(f, filedata("readme.txt", "This is some text"))
